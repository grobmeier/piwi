getCustomSiteMap

public static The getCustomSiteMap(string id, integer depth, boolean includeParent)

Returns the 'SiteMap' which is an array of SiteElements representing the website structure. It is possible to retrieve only certain parts of the 'SiteMap'.

Parameters:
id - Returns only the SiteElement with this id and its subitems. Set to 'null' if all SiteElements should be returned.
depth - Determinates the maximum number of subitems that will be returned. Set to '-1' for full depth, to '0' only for parent items.
includeParent - If false only the children of the parent will be returned. This can only be used if $id is not 'null'.
Returns:
'SiteMap' which is an array of SiteElements representing the website structure.
