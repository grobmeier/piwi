private class ezcBaseWhateverException
extends ezcBaseException

ezcBaseWhateverException is thrown whenever something is so seriously wrong. If this happens it is not possible to repair anything gracefully. An example for this could be, that your eZ components installation has thrown far to many exceptions. Whenever you receive an ezcBaseWhateverException, do not even try to catch it, but forget your project completely and immediately stop coding! ;)
