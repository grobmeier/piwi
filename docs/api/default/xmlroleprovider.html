isUserInRole

public boolean isUserInRole(string username, array roles)

Checks if a user has one of the given roles. If the user has at least one of the given roles, true is returned, otherwise false.

If a user is not authenticated and the roles array contains the role "!", the user is in role.

In other terms, if a page has the ! role, it can only be viewed by unauthenticated users.

Parameters:
username - The name of the user.
roles - The roles that are allowed.
Returns:
True if user has at least one of the given roles, otherwise false.
